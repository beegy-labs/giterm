## M1: SSH Terminal (Q1 2026)

Single SSH connection terminal with basic connection management.

- SSH connection via password/key authentication
- xterm.js terminal rendering with WebGL
- Connection save/load (localStorage)
- Desktop + iOS support

## M2: Multi-tab + Connection Manager (Q2 2026)

- Multiple simultaneous SSH sessions
- Tab-based terminal switching
- Connection groups and folders
- SSH config file import

## M3: Agent Integration (Q3 2026)

- AI-assisted terminal commands
- Command suggestions and autocomplete
- Session recording and replay

## M4: Orchestration (Q4 2026)

- Multi-server command execution
- Workflow automation
- Template-based operations
